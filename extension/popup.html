Quick Setup

Three steps to connect your browser to Cursor

1 Install the native host

Run once in your terminal

2 Start the MCP server 
npx brms-host serve

Keep this running in a terminal tab

3 Add to Cursor MCP config 
{
  "mcpServers": {
    "brms": {
      "url": "http://localhost:3100/mcp"
    }
  }
}

Paste into .cursor/mcp.json in your project root
