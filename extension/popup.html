Quick Setup

Two steps to connect your browser to Cursor

1 Install the native host
npx brms-host install
2 Add to Cursor MCP config
{ "mcpServers": { "brms": { "url": "http://localhost:3100/mcp" } } }

Paste into .cursor/mcp.json in your project root
